Primer paso
----------------------------
Correr el comando 
### `npm run installAll`
para instalar las dependencias


Segundo Paso
----------------------------
Ejecutar 
### `npm run dev`
para correr el servidor


Tercer Paso
----------------------------
Abrir otra consola en el directorio Client o bien en la misma carpeta 
### `cd Client`
### `npm start`
para correr el cliente
